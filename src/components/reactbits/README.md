# ReactBits components

Paiguta siia `react-bits` komponendid, mille lisad käsuga:

```bash
npx jsrepo add github/DavidHDev/react-bits <component-name>
```

## Kasutus selles projektis

Sektsioonid importivad komponente siit kaustast. Kui mõni komponent pole veel lisatud,
kasutab projekt selle jaoks fallback'i (lihtne framer-motion / CSS efekt), et build ei katkeks.
